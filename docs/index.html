    Targeted platforms

    QMock doesn't rely on host objects such as the DOM, or BOM, and as a CommonJS-compliant module can be run on the server. It targets the following engines:

    • Microsoft JScript (IE6+)
    • Mozilla SpiderMonkey [versions] (Firefox 1.5 and higher)
    • Webkit JavaScriptCore [versions] 2.0.4 and higher
    • Opera [presto?] 9.25 and higher
    • Chrome V8

    Testrunners

    QMock has also been designed to be compatible with as many testrunners as possible, so existing test suites can easily be extended to include integration, or 'glass-box' tests. The only requirement being that the runner, such as QUnit, Evidence, or the CommonJS Assert module (to name a few) expose some kind of comparison method (e.g. QUnit.equiv, assert.deepEquals) which QMock uses to test the received parameters against the mock object expectations.
