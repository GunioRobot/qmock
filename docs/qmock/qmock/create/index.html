    class method QMock.create 

    QMock.create() → QMock instance

    Factory method for instantiating a new QMock instance with separate state and copies of private/public members

    This is potentially useful for running two versions side-by-side with custom comparison methods, or modifiying any of the public members.

    Example

    QMock.create(); // Object (API)
