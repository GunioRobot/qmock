    class method QMock.utils.verify 

    QMock.utils.verify( receiver [, raise] ) → Boolean | Exception
    • receiver (Mock): mock / receiver object to test
    • raise (Function) optional: Function to handle false comparison results

    Verifies the receiver object (the parent mock object) first, then individual members (both methods and nested namespaces).

    Only passes if whole object tree passes, else throws exception or returns false depending on mode (fail slow/fast).
