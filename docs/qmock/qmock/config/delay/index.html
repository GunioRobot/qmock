    class property QMock.config.delay 

    QMock.config.delay → 100 | Number

    Configuration for the delay on Ajax callbacks. This is used to help simulate asynchronous transactions and the latency of a round-trip to a server. In turn this encourages developers to developers to ensure callbacks enact upon the right data or view when executed.
