    instance method Receiver#verify 

    Receiver#verify( [raise] ) → Boolean

    Method recurses through all the child mock object members of a receiver mock object, verifying their interactions. It delegates the actual instance verification routine to Mock#verify. It will returns a Boolean, wher an overall result is determined by whether the receiver, or any 'leaves' had a negative result.

    It will also throw an error if optional raise parameter is passed. This is thrown if QMock.config.failfast is set to true. It can also be retrieve by calling Mock.__getExceptions().

    See Mock#verify for mock object verification details.
