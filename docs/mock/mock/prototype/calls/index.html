    instance method Mock#calls 

    Mock#calls( min [, max = null] ) → Mock
    • min (Number): Miniumum number of times mock object should be invoked. If max parameter not passed then number becomes a 'strict' invocation expectation (even zero). Default is null.
    • max (Number) optional: Maximum number of times mocked method should be called. If want 'at least N' then just pass Infinity. Default is null

    This method mainly used to set invocation expectations on constructors / receiver objects, as conventional usage for mock methods is to pass parameters to Mock.expects.

    However, Mock.calls can be called on an instance of Mock.

    Examples

    // Set expected calls on mock constructor / receiver
Mock.calls(1,5);

// Set calls on mockmethod
Mock.expects().method('foo').calls(1,5);
