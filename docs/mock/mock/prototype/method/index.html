    instance method Mock#method 

    Mock#method( name ) → Mock
    • name (String) – Name of the method

    When method() is called on a receiver object it is augmented with a new method bound to the identifier name.

    Throws InvalidMethodNameException if member with key name already exists on receiver.

    Example

    Mock.expects().method('foo');
